About Me

I'm a developer, residing in Mexico. I started learning programming back in 2010. Since then I've worked on games, tools, cloud servers and websites.

My introduction to programming was fueled by my love for gaming, when I was younger I remember being so amazed by the virtual worlds from my favorite games that I decided I wanted to learn how to code, which lead me to discover my passion for creating stuff.

Lately I've been learning about music and audio software, my goal is to gain the skills needed to make music for my own projects.

Tools

This section shows some tools I have worked on, mostly focused to gamedev and audio.

Platformer 2D System (Unity)

A collision detection and movement system for 2D platformers, the goal of this project is to provide core functionality to help with the creation of custom character controllers.

Source Code

Automatic Grass Tool (Unity)

Made for Smashing Crystals to save time during level creation, the tool automatically generates grass based on the terrain's normal. The code can be viewed in the game's submission article.

Zombie Pattern Editor (Unity)

Made for Undead Masher, this editor allowed the game designer to quickly create new zombie patterns in-game via a custom scripting language.

Yamaha S-YXG50 Control Panel (JSFX)

A Reaper JSFX midi effect that allows to control the Yamaha S-YXG50 synth, this is a fork from Shiru's control panel.

Source Code

DMW Converter (.NET)

Command line tool to convert DefleMask binary wavetable files (.dmw) to human readable text files.
