Tools

This section shows some tools I have worked on, mostly focused to gamedev and audio.

Rope/Chain Simulator (Unity)

Ideal for character clothes such as scarfs and capes, uses the Verlet Integration method, the simulation runs at a fixed timestep (with interpolation) and features an override/preset system.

Source Code

Platformer 2D System (Unity)

Collision detection and movement system for 2D platformers, the goal of this project is to provide core functionality to help with the creation of custom character controllers.

Source Code

Zombie Pattern Editor (Unity)

Made for Undead Masher, this editor allowed the game designer to quickly create new zombie patterns in-game via a custom scripting language.

Yamaha S-YXG50 Control Panel (JSFX)

Reaper plugin that allows music producers to control the Yamaha S-YXG50 synthesizer. This is a fork from Shiru's control panel to which I added many new features, UI improvements and bug fixes.
